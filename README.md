# Kwikwi bot

## Installation

``` bash
$ make install
$ vim config.js
```

```js
module.exports = {
  "apiKey": "yourDiscordAppSecretKey",
  "activity": "what you want",
  "admins": [
    "yourLogin#5555"
  ]
}
```

```bash
$ make start
```

Ajoute le bot dans ton serveur discord et tape `/kwikwi help`.
